# MondayBot

Versions:
- Check `requirements.txt` for module versions
```
$ python3 --version
3.7.3
```

Package installation:
```
$ python3 -m pip install --upgrade pip
$ python3 -m pip install -r requirements.txt
```

Run the bot:
- Get `key.txt` from me and place in the project directory
```
$ python3 bot.py
```
